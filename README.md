# formulario-de-alistamento
Projeto desenvolvido em HTML como exercício de formulários. O sistema simula o cadastro de candidatos para uma missão espacial civil, contendo campos para informações pessoais, login, especialidade, 
treinamentos, justificativa para participação na missão e aceite dos termos de risco. Projeto criado para praticar formulários, validações, radio buttons, checkboxes, textarea e organização de dados
com fieldsets.
